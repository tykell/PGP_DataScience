To test the project:

1. Install all the required libraries/ packages.

2. Create a folder named "data", and create two subfolders in it: "after_eda" and "fea_sample".

3. In the "data" folder, upload the train.csv, test.csv, bfs_benchmark.csv, and random_benchmark.csv

4. Download the python files and paste them along with the "data" folder.

5. Using the Jupyter notebook, execute the .ipynb files in the following order:
	  (i) Data cleaning and EDA
	 (ii) Data preprocessing and data preparation for ML model
	(iii) Graph Link Prediction
